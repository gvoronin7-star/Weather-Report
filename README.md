# Weather Report 🌤️

Консольное и графическое приложение для отображения текущей погоды по названию города или координатам.

![Python](https://img.shields.io/badge/python-3.14+-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Tests](https://img.shields.io/badge/tests-19%20%F0%9F%9A%80-brightgreen)

## Возможности

- 🔍 **Поиск по городу** — введите название города и получите текущую погоду
- 📍 **Поиск по координатам** — укажите широту и долготу
- 🖥️ **Два интерфейса** — CLI (консольный) и GUI (графический)
- 💾 **Кэширование** — сохранение последнего запроса для работы офлайн
- 🔄 **Ретраи** — автоматические повторные попытки при сетевых ошибках
- 🌍 **Множественные результаты** — выбор города из списка при совпадении названий

## Установка

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/yourusername/weather-report.git
cd weather-report
```

2. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

3. **Получите API ключ:**
   - Зарегистрируйтесь на [OpenWeatherMap](https://openweathermap.org/)
   - Перейдите в раздел [API Keys](https://home.openweathermap.org/api_keys)
   - Скопируйте ваш ключ

4. **Настройте конфигурацию:**
```bash
# Отредактируйте файл .env
# Замените your_api_key_here на ваш ключ
API_KEY=your_api_key_here
```

## API

Приложение использует бесплатные API OpenWeatherMap:

| API | Endpoint | Описание |
|-----|----------|----------|
| Geocoding | `geo/1.0/direct` | Преобразование города в координаты |
| Weather | `data/2.5/weather` | Текущая погода по координатам |

## Конфигурация

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `API_KEY` | API ключ OpenWeatherMap | обязательно |
| `CACHE_FILE` | Файл кэша | `weather_cache.json` |
| `CACHE_MAX_AGE_HOURS` | Макс. возраст кэша (часы) | 3 |
| `MAX_RETRIES` | Макс. попыток при ошибке | 3 |

## Ошибки и решения

| Ошибка | Причина | Решение |
|--------|---------|---------|
| `API_KEY не найден` | Не настроен .env | Добавьте ключ в .env |
| `Город не найден` | Город не существует | Проверьте название |
| `Превышен лимит запросов` | Too Many Requests | Подождите и повторите |
